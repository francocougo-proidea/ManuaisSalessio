Novo

Neste menu é possível escolher o Produto que você deseja configurar, e filtrar os produtos por Modelos e Coleções

1° Coleções

As coleções irão filtrar os Modelos e Produtos.

Ao escolher uma Coleção, só irão aparecer os Modelos que fazem parte da mesma.

Todos os produtos que fazem parte desses Modelos irão aparecer também.

Clique aqui para ver como criar uma Coleção

Novo 1.1

2° Modelos

Os modelos irão filtrar os Produtos.

Ao escolher um Modelo, só irão aparecer os Produtos que fazem parte do mesmo.

Clique aqui para ver como criar um Modelo

Novo 1.2

3° Pesquisa

A barra de pesquisa serve para buscar um Produto pelo nome.

Ela ignora os filtros de Coleção e Modelo, mostrando todos os Produtos que possuem aquela palavra-chave no seu nome.

Novo 1.4

3° Seu Carrinho

Após configurar um produto e clicar em Salvar, o mesmo produto será adicionado ao seu carrinho. 

Clicando no botão do carrinho, podemos ver todos os itens que temos adicionados, caso não tenha nenhum item, uma mensagem irá aparecer dizendo que o carrinho está vazio.

Orçamento

Ver/Editar o orçamento do pedido selecionado, incluindo os dados do cliente, informações complementares, opções de pagamento, status e muito mais.

Como temos muitas opções, vamos separá-las por grupos para explicar melhor cada uma.

Dados do Orçamento

1° Opções Pré-Definidas: Essas opções são definidas automaticamente pelo sistema.
2° Icms: Selecione a porcentagem de imposto ICMS que será aplicada.

Dados do Cliente

1° Cadastrar Cliente

 Clicando neste botão, você pode cadastrar um cliente novo.

Dados: 
    Código: Preenchido automaticamente.
    Ativo: Define se o cliente está ativo ou não.
    Descrição: Descrição do cliente.
    Tipo Pessoa: CPF ou CNPJ
    CPF/CNPJ: Digitar de acordo com o Tipo Pessoa.

  Dados Complementares: Cadastro básico e opcional.


2° Selecionar Cliente

 Clicando neste botão, você pode selecionar um Cliente para atribuir a este pedido.
  

3° Editar Cliente

Clicando neste botão, você pode editar os dados do cliente selecionado.
  

Dados Complementares

1° Descrição: Inserir uma descrição para o pedido. 
2° Ordem de Compra: Inserir uma ordem de compra para o pedido.
3° Ambiente: Digitar um ambiente onde o produto será usado.
4° Transportadora: Selecionar a Transportadora
5° Observação: Digitar uma observação para o pedido

Opções de Pagamento

1° Pagamento: Selecionar como será o pagamento (56, A VISTA, ANTECIPADO, 28 DIAS)
2° Divisão de venda: Selecionar a divisão de venda (VENDA, AT, SHOWROOM, VENDA EXTERIOR)
3° Tipo de desconto: Selecionar o Tipo de Desconto (SEM DESCONTO, PERCENTUAL, COEFICIENTE, VALOR)
4° Tipo de acrescimo: Selecionar o Tipo de Acrésicmo (SEM ACRESCIMO, PERCENTUAL, COEFICIENTE, VALOR)

Status

1° Status: Selecionar o Status do pedido (ORÇAMENTO, PEDIDO CONFIRMADO, PRODUZINDO, EM PROCESSAMENTO,  CONCLUIDO, RECUSADO, CANCELADO)
2° Botão Seta: Atualizar o orçamento com o status corrigido.

Produto Configurado

1° Símbolo de Olho

Aqui você pode acessar o menu de configuração do produto selecionado para alterá-lo.

2° Símbolo de Seta

Aqui você pode fazer upload de uma imagem, ou arquivo para seu pedido.

3° Símbolo de ( i )

Aqui você pode ver/editar detalhes do seu pedido como:

    - Código Cliente
    - Ambiente
    - Observação
    - Tipo de Desconto
    - Tipo de Acréscimo

4° Símbolo de $

Ver a estrutura de preço, mostra todos os itens que serão cobrados no seu produto.

É possível clicar no botão "Exportar CSV" para gerar um arquivo .csv com as informações dessa lista.

5° Símbolo de Lista

Ver a estrutura de produto do seu pedido, mostrando todos os itens que irão compor o produto.

É possível clicar no botão "Exportar CSV" para gerar um arquivo .csv com as informações dessa lista.

6° Símbolo de Impressora

Gera um .PDF para impressão com as informações do seu produto, e um desenho 2D do mesmo.

7° Símbolo de Ficha

Mostra um LOG contendo todas as alterações que foram feitas no seu Orçamento até o momento.

É possível clicar no botão "Exportar CSV" para gerar um arquivo .csv com as informações dessa lista.

Botões extras

1° Duplicar: Duplica este orçamento

2° Cancelar: Altera o status deste orçamento para "cancelado"

3° Imprimir Orçamento: Gera um pdf para impressão contendo informações do orçamento

4° Imprimir Listagem: Gera um pdf para impressão contendo uma lista dos produtos neste orçamento

5° Log Alterações: Abre uma janela que mostra as alterações realizadas neste orçamento

6° Log calculo Preço: Abre uma janela que mostra como os descontos, acréscimos, impostos e mais influenciam o preço final.

7° Baixar xml erp: Abre em uma nova janela um arquivo em formato xml contendo todas informações do seu orçamento.

8° Gerar Excel: Gera um arquivo .xlsx para baixar, contendo algumas informações do seu orçamento.

9° Imprimir Todas as fichas: Gera um arquivo pdf para download com informações detalhadas de todos os pedidos do orçamento.

10° Imprimir Pedido de Compra: Gera um pdf para impressão contendo os pedidos de compra deste orçamento.

11° Baixar todos os anexos: Baixa todos os arquivos que foram anexados em qualquer pedido deste orçamento.

12° Imprimir Etiquetas: Imprime as etiquetas do orçamento.

Configuração de Produto

Este menu é dedicado a configurar o Produto que você selecionou, escolhendo as suas dimensões e características.

Como temos muitas opções, vamos separá-las por grupos para explicar melhor cada uma.

Dados do Orçamento

1° Seu Carrinho: Acessar seu carrinho de compras. Se estiver vazio um aviso será mostrado na tela.

Dimensões

1° Medida Especial?: Altera as dimensões máximas e mínimas de largura, e altura, e profundidade para uma medida especial definida na criação do produto.

2° LARG: Digite uma Largura para o produto.

3° ALT: Digite uma Altura para o produto.

4° PROF: Digite uma Profundidade para o produto.

Características

As características são perguntas que o usuário responde para configurar o produto de acordo com suas preferências

Características Normais

As características normais são simples.

Você clica nela, e escolhe a opção que melhor se adapta ao produto que deseja criar.

Característica Divisor

A característica do Divisor é um pouco mais complexa. 

Com ela você pode adicionar os divisores verticais
(Colunas, etc), e os divisores horizontais também
(Prateleiras, etc).

Além disso, você consegue editar a posição de cada divisória individualmente.

Informações Complementares

Informações opcionais que serão salvas no seu orçamento. Todas as informações complementares são opcionais.

1° Observação: Informar uma observação para o pedido.
2° Ambiente: Informar um ambiente onde o pedido será utilizado.
3°Cod.Cliente: Informar um código para o cliente.
4° Quantidade: Informar a quantidade de produtos iguais a esse que deseja por no orçamento.

Últimos botões

1° Desenhar

Gera uma imagem 2D do produto configurado.

IMPORTANTE: É necessário desenhar o produto para poder salvá-lo.

2° Sair Sem Salvar

Sai da configuração de produto sem salvá-lo antes.

3° Salvar

Salva o produto configurado no carrinho e volta para a tela inicial.

Importante: Para ter certeza que o produto configurado está salvo, é necessário clicar no carrinho logo após voltar para a tela inicial, igual demonstrado na imagem.

4° Salvar, Duplicar e Editar

Salva o produto no carrinho, duplica-o e automaticamente já entra no modo de configuração no pedido duplicado.

5° Log Limitações

No sistema Saléssio é possível adicionar limitações por produtos, por características, por usuários e por grupos de usuários.

No log de limitações é possível ver todas as limitações que estão afetando este produto.

6° 3D

Clicando neste botão, um Modelo 3D do produto é gerado.
