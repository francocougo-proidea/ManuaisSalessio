Cadastro

Os cadastros não chegam a mexer no catálogo de produtos, para isso, veja os manuais de catálogo, os cadastros servem para cadastrar Tabelas de Acréscimo, Atendentes, Categorias de Itens, Usuários e Grupos de Usuários e muito mais.

Cabeçario

Opções de Cadastro

Tabelas - Usadas para fins administrativos e gestão.
Exemplo: Acréscimos, Descontos, Categorias de Itens... 

Grupos de usuários - Separar usuários por grupos, de modo que ao realizar limitações no grupo, elas sejam aplicadas a todos os usuários do grupo.

Usuários - Criar ou editar um usuário do sistema.

Notícias - Publicar notícias que aparecem na Tela Inicial.

Política Compra - Informações genéricas que vão aparecer em todos os orçamentos.

Pessoas - Escolher um tipo de usuário entre Cliente, Fornecedor e Transportadora. Todos os usuários do tipo pessoa estão vinculados ao usuário de criação.

Markup dos Clientes - Definir o valor extra que um usuário deverá pagar.

Políticas descontos - Criar políticas de desconto que podem ser aplicadas nos usuários.

Alterar status Pedidos Manual - Alterar o status pedido.
Exemplo: Confirmado, em andamento...

Abaixo vamos explicar com mais detalhes cada um dos menus mencionados acima.

Tabelas

Você pode criar uma tabela nova ou editar uma tabela existente.
Existem diversas tabelas para diversos fins diferentes, mas a maioria delas é para fins administrativos e de gestão.

Importante: As tabelas que já vem cadastradas não podem ser excluídas, pois elas fazem parte de outras gestões e essa exclusão poderá gerar erros no sistema.

Tabelas 1.0

Interface Inicial

1° Cadastrar nova tabela: Clique no botão para ir para a janela onde você cadastra uma nova tabela no sistema.

2° Código: Filtrar as opções pelo código e organizá-las em ordem alfabética.

3° Descrição: Filtrar as opções pela descrição e organizá-las em ordem alfabética.

4°Código ERP: Filtrar as opções pelo código ERP

Você pode editar uma tabela existente clicando em cima da opção dela.

Para cadastrar tabela, clique em Cadastrar Nova Tabela ( 1 )

Tabelas 1.1

Cadastrar Nova Tabela

1° Codigo: Código de identificação da Tabela
2° Descrição: Descrição da Tabela
3° Tipo de Característica: Inalterável
4° Car. Referencia: Copia as Informações de outra Tabela.
5° Tipo Preview: Se a apresentação dessa tabela vai ser com imagens ou em formato de lista.
6° Campos Adicionais: São os campos que podemos incluir na Tabela.

Tabelas 1.2

7° Importar txt: Importa um arquivo .TXT com as opções da Tabela
8° Preview: Visualizar como as opções da tabela apareceram para o usuário
9° Log: Mostra as alterações que foram feitas na Tabela Atual
10° Exportar txt: Exporta um arquivo de texto contendo todas as opções da tabela. É possível Edita-lo em um editor de Planilha, e depois importa-lo novamente para o Saléssio  usando o botão Importar txt (7°)
11° Categorias: Criar categorias para separar as opções.
12° Excluir: Excluir a Tabela atual.
13° Salvar: Salvar o Cadastro da Tabela. (Isso não salva as opções que você criar dentro da tabela, somente o cadastro da tabela) 


Grupos de Usuários

Aqui você pode criar grupos para separar os usuários do sistema.
As limitações e mudanças aplicadas a um grupo serão aplicadas também a todos os usuários pertencentes a este grupo.

Importante: As limitações por usuário são superiores às limitações do grupo. Exemplo: Se no grupo Fabricante todas as cores de perfil são liberadas, mas o usuário X possui uma limitação de cores no seu usuário, essa limitação terá superioridade sobre as limitações do grupo.

Grupos de Usuários 1.0

Interface Inicial

1° Cadastrar Grupo de Usuários: Clique para criar um novo Grupo de Usuários.
2° Codigo: Filtrar as opções pelo código e organizá-las em ordem alfabética.
3° Descrição: Filtrar as opções pela descrição e organizá-las em ordem alfabética.

Grupos de Usuários 1.1

Cadastro de Grupos de Usuários

1º Codigo: Código de Identificação do Grupo.
2° Descrição: Descrição do Grupo.
3° Política de desconto: Política de desconto usada pelo Grupo.
4° Limitações: Limitar características que podem ser utilizadas pelo Grupo.
5° Limitações de produto: Limitar produtos que podem ser utilizados pelo Grupo.
6° Excluir: Excluir Grupo de Usuários.
7° Salvar: Salvar Grupo de Usuários.

Grupos de Usuários 1.3

Limitações de Característica

1° Selecionar a característica: Selecione a característica que deseja limitar para esse grupo.
2° Opção: Selecione as opções da característica que estarão ativas para este grupo.
3° Salva: Salva as limitações feitas na característica selecionada.
4° Excluir: Excluir as limitações feitas na característica selecionada.
5° Seleciona Todas: Ativa todas as opções da característica selecionada.
6° Deseleciona Todas: Desativa todas as opções da característica selecionada.
7° Fechar: Feche a janela de Limitações.

Grupos de Usuários 1.4

Limitações de Produto

1° Produtos: Selecione os Produtos que serão desativados para este Grupo de Usuários.

2° Fechar: Feche a janela de limitações de produto.

3° Salva: Salva as limitações de produto feitas neste Grupo.

4° Deseleciona Todas: Retira todas as limitações de produto do grupo.

Usuários 1.1

Cadastrar novo usuário

1° Ativo: Definir se o usuário está ativo ou não.
2° N° Max de Vendedores: Seleciona o n° máximo de vendedores que este usuário pode ter.
3° Codigo: Código que será usado para acessar o sistema.

5° Senha: Senha que será usada para acessar o sistema.

7° E-mail: Email que será utilizado para a recuperaçã de senha, caso a esqueça.

8° Tipo:
Você poderá selecionar entre:
 -Administrador: Possui permissão para criar e/ou editar qualquer cadastro ou senha. Sendo o único usuário que possui acesso as configurações do sistema.
 -Atendente Interno: Visualizará os pedidos do cliente que tem ele mesmo como atendente.
 -Representante: Visualizará os pedidos de clientes que tem ele como representante.
 -Atendente Geral: Ele poderá acessar todos os orçamentos/pedidos do sistema, a aba de cadastros, menos as configurações do sistema e do catálogo.
 -Cliente: Poderá ser uma loja, um cliente pessoa física
 -Vendedor: Visualizará somente os orçamentos feitos por eles.
 -Vendedor conferente: Visualizará os orçamentos realizados por ele e os orçamentos realizados por todos os vendedores cadastrados pelo mesmo pai.

Notícias

Aqui você pode cadastrar notícias e informativos para apresentar na tela inicial.

Notícias 1.0

Interface Inicial

1° Cadastrar nova Notícia: Clique aqui para cadastrar uma nova Notícia.

2° ID: Filtrar as opções pelo seu ID.

3° Descrição: Filtrar as opções pela sua descrição e organizá-las em ordem alfabética.

4° Tipo: Filtrar as opções pelo Tipo e organizá-las em ordem alfabética.

Notícias 1.1

Alterar/Cadastrar Notícias

1° ID: É preenchido automaticamente e de forma sequencial.
2° Descrição: Definir uma descrição para o informativo/notícia.
3° Data Início: Define a partir de qual data o informativo/notícia começará a ser apresentado.
4° Data Fim: Define o dia em que esse informativo/notícia não será mais apresentado.
5º Título: Define o título do informativo/notícia.
6° Imagem: Define a imagem com a informação que será apresentada.

7° Tipo: Poderá ser Noticia ou Informativo:
    -Notícia: Será apresentada abaixo do banner informativo na tela inicial.
    -Informativo: Aparecerá quando o usuário fizer login.
    -Informativo (somente like/unlike): Aparecerá quando o usuário fizer login. O usuário deverá selecionar essa informação como positiva ou negativa. O informativo será mostrado até que o usuário selecione uma opção.

8° Notícia: Escrever a notícia que será apresentada.
9° Salvar: Salvar a notícia.
