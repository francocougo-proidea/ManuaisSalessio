Alterar Markup

Essa opção não está disponível para usuários do tipo Vendedor.

Ao acessar, será solicitado que você informe a sua senha de markup. Caso o lojista optar por não ter o markup da loja cadastrado, não será necessário nenhum cadastro.

Após clicar em Entrar, você é redirecionado para a seguinte janela:

Alterar Markup 1.0

Interface Secundária

1° Vendedor: Você poderá selecionar um vendedor para aplicar markup ou, caso não seja selecionado um vendedor específico, o markup será aplicado a todos os vendedores.

2° Markup Geral: Informa um valor;

3° Tipo: Define o tipo de Markup: Porcentagem ou Coeficiente
    3.1° Porcentagem: Aplica um incremento em porcentagem que é calculado pelo sistema e aplicado ao produto/orçamento.
    Exemplo: usando um markup de 5% no valor de um produto de R$80,00 o valor final para o cliente será R$ 84,00;
    3.2° Coeficiente: Aplica um incremento por coeficiente onde o sistema multiplica o valor escolhido e aplica ao produto/orçamento.
Exemplo: usando um coeficiente de 4 os preços de compra serão multiplicados por 4, ou seja, R$ 80,00 com coeficiente 4 ficará um valor total de R$ 320,00.

Lista de Vendedores

Nessa opção aparecerá a lista de vendedores cadastrados pelo seu usuário (por usuários-pai).

1° Cadastrar Vendedor: Clique para ir para a janela de cadastro de vendedores
2° Código: Filtrar vendedores pelo seu código
3° Descrição: Filtrar vendedores pela sua descrição

Lista de Vendedores 1.0

Cadastro Vendedor -
Dados Principais

1° Codigo: Automaticamente preenchido.
2° Descrição: Nome/Login para acesso ao sistema.
3° Ativo: Selecione se o Vendedor está ativo ou não.
4° Preço de compra: Define se o usuário criado poderá visualizar o preço de custo.
5° Permissões de desconto: Seleciona se o Vendedor pode aplicar descontos, ou não.
6° Senha: Define senha para acesso ao sistema.
7° E-mail: Utilizado para recuperação de senha.
8° Cod Pai: Preenchido automaticamente.
9° Tipo Vendedor: Você deve selecionar entre Vendedor, ou Vendedor Conferente:

    9.1° Vendedor: Visualizará somente os orçamentos feitos por eles.
    9.2° Vendedor conferente: Visualizará os orçamentos realizados por ele e os orçamentos realizados por todos os vendedores cadastrados pelo mesmo pai.

Cadastro Vendedor - 
Contato

1° Cadastro Básico: Aqui você vai realizar um cadastro básico. Lembrando que esses campos são opcionais.

2° Salvar: Salvar as alterações feitas nesse Vendedor.
